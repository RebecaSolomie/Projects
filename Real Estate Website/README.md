# Real Estate Website

A real estate platform that allows users to browse available properties and enables agents to manage property listings. Built with React and Node.js, containerized with Docker, and deployed for production use.

## Features

### For Visitors
- View all available estates with detailed information.
- Search and filter properties by location, price, type, and more.

### For Real Estate Agents
- Register and log in securely.
- Add new property listings.
- Edit existing property information.
- Delete property listings.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express.js
- Database: PostgreSQL
- Authentication: JWT
- Containerization: Docker
- Deployment: Cloud server
